# AI-Assignments
This repository contains simple implementations of different Artificial Intelligence concepts in C.

## Contents

### AQI Agent
Implements a simple reflex agent that calculates the Air Quality Index (AQI) using pollutant data from a file.

### CAPTCHA
A basic simulation of a text CAPTCHA system used to distinguish humans from automated bots.

### Turing Test
A simple program demonstrating the concept of the Turing Test, where responses may come from either a human or an AI.

### Uninformed Search
Implementation of Breadth First Search (BFS) and Depth First Search (DFS) applied to the Water Jug Problem.  
The program also compares the performance of BFS and DFS.
